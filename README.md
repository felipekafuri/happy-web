## How to run the project? 

**Clone**
```bash
git clone https://github.com/felipekafuri/happy-web
```
**Va até o diretório** 
```bash
cd happy-web
``` 

**Install the dependencies**
```bash
yarn 
``` 
or
```bash
npm install 
``` 

**Start the project**
```bash
yarn start
```
